## Descarga
    Desde una consola o el bash de git usar el siguiente comando
        git clone https://github.com/juanbaquiroga/AMATE.git


## Instalacion:
    * pip install -r requirements.txt
    * python manage.py migrate
    * python manage.py runserver
    
# secciones
  ## para visitar la pagina debemos ingresar a http://127.0.0.1:8000/home/
  - en esta seccion podremos ver las publicacines y agregar nuevas tocando el boton 'agregar publicacion'
  - tocando en 'ver mas' accedemos al detail de la publicacion donde podremos editarla o eliminarla
  
  ## seccion 'productos'
  - en esta seccion podremos ver los productos
  - tocando en ver mas accedemos al detail del producto, donde podremos editarlo o eliminarlo
  
  ## seccion 'about us'
  - aqui podran encontrar informacion acerca de como surge nestro sitio
  
  ## seccion 'login' 
  -en esta seccion los usuarios ya registrados pueden acceder a su usuario
  
  ##seccion 'register' 
  -en esta seccion los usuarios podran registrarse en el sitio
  
  ## barra de busqueda
  - este buscador nos redirige a la pagina de productos y nos muestra los productos que coinciden con lo buscado
  
  
# una vez registrados
  
  
  ## seccion 'crear producto'
  - en esta seccion podremos agregar nuevos productos

  ## seccion 'categorias'
  - en esta seccion podremos ver las categorias existentes y editarlas o eliminarlas
  - tocando en 'agregar categoria' podremos agregar una nueva categoria para los productos
  
  ##*nombre de usuario*
  -accediendo aqui podremos editar nuestro perfil(foto de perfil, telefono y web) y nuestro usuario(sername y email)
  
  ##seccion logout
  -al hacer click en este boton podremos cerrar nuestra sesion
